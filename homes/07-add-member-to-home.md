# Add Member to Home

Add member to home by home Id and user Id.

**URL** : `{endpoint}/v2/homes/member/add/:homeId/:uid`

**Method** : `POST`

**Auth required (Bearer token)** : YES

**URL Parameters** : `homeId = [desc: Home Id, type: string]` and `uid = [desc: User Id, type: string]`


## Success Response

**Code** : `204 No Content`
